# Cookie Bot

normal version

## 사용 방법
1. 오른 쪽 위. clone 버튼 클릭
2. download zip 클릭
3. 압축 해제.
4. 폴더 들어가서. 우 클릭 vscode로 열기
5. npm i
6. npm run start **bot token**
